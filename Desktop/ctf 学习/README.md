# CTF 学习导航 🦞

风雨丛中笑，兄弟心中照 —— 一站式 CTF 学习导航页。

## 项目结构

```
ctf 学习/
├── index.html                  # 主页面（CTF 导航 + 三大指令手册）
├── css/
│   ├── style.css               # 主样式（暗色主题）
│   └── mascot.css              # 吉祥物样式
├── js/
│   └── main.js                 # 交互逻辑（热力图、搜索、手册筛选等）
├── kali-指令速查手册.html       # Kali Linux 安全工具速查（独立页面）
├── linux-指令教学.html          # Linux 命令行入门教学（独立页面）
├── git-指令速查手册.html        # Git 版本控制速查（独立页面）
└── README.md
```

## 功能模块

### 🏠 首页导航
- **CTF 题型速查** — Web/Misc/Crypto/Reverse/Pwn 五大方向介绍
- **菜鸡路线规划** — 从零开始的 6 个月学习路线（支持勾选进度，localStorage 持久化）
- **CTF 训练平台** — BUUCTF / BugKu / 攻防世界 / CTFHub / 青椿 CTF
- **安全社区** — FreeBuf / 先知社区 / 安全客 / Seebug 等（支持搜索）
- **Web 技术演化** — 折叠式时间线（简单网站 → 数据交互 → 架构演进 → 云服务 → 软件开发）
- **练手项目** — 鱼皮 ceshiya 网络安全自学网（本地启动指引）
- **内置工具箱** — Base64 / Hex / ROT13 / URL 编解码 + 哈希自动识别
- **CodeFather** — 程序员鱼皮编程导航

### 📊 数据追踪
- **GitHub 风格热力图** — 记录每日点击，数据存 localStorage
- **站点排行榜** — Top 10 最常访问站点
- **CTF 赛事倒计时** — 自动计算距下一场比赛的时间
- **数据导入/导出** — JSON 格式备份恢复

### 📖 三大指令手册（已集成入 index.html）
| 手册 | 分类 | 指令数 | 特点 |
|------|------|--------|------|
| 🐧 Linux 指令学院 | 10 | 54 | 文件操作、权限、进程、网络、文本处理等 |
| ☠️ Kali Linux 速查 | 10 | 56 | 信息收集、漏洞分析、Web 攻击、密码攻击等 |
| ⎇ Git 速查手册 | 10 | 50+ | 基础操作、分支、远程、撤销、协作等 |

每个手册支持：
- 分类标签筛选
- 实时关键词搜索
- 点击卡片展开语法/示例/提示

## 使用方式

直接双击 `index.html` 或在浏览器中打开即可，无需构建工具。

所有数据（点击记录、路线图进度）存储在浏览器 localStorage 中。

## 吉祥物

🦞 丛中笑 — 点击可互动，会随机说鼓励的话、给学习建议。每 5 分钟自动冒泡一次。
