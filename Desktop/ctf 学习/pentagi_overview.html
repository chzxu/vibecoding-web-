03 · Architecture

Modular, Scalable, Secure

Microservices-based design with clear separation of concerns — built for production deployments.

Core Services

🤖 AI Engine
  • Frontend UI: React + TypeScript web interface
  • Backend API: Go-based REST & GraphQL
  • Vector Store: PostgreSQL + pgvector
  • Task Queue: Async processing system
  • AI Agent: Multi-agent system with specialized roles

Knowledge

🧠 Memory & Graph
  • Graphiti: Knowledge graph API for semantic relationships
  • Neo4j: Graph database for entity relationships
  • Long-term Memory: Persistent experiences & patterns
  • Working Memory: Active context & goals
  • Episodic Memory: Historical actions & outcomes

Monitoring

📈 Observability
  • OpenTelemetry: Unified data collection
  • Grafana: Real-time dashboards & alerts
  • VictoriaMetrics: High-performance metrics
  • Jaeger: Distributed tracing
  • Loki: Log aggregation & analysis

Analytics

📊 LLM Analytics
  • Langfuse: LLM observability & analytics
  • ClickHouse: Analytics data warehouse
  • Redis: Caching & rate limiting
  • MinIO: S3-compatible artifact storage
  • Chain Summarization: Intelligent context management

Security Tools

🛡️ Tool Execution
  • 20+ Professional Pentesting Tools (nmap, sqlmap, metasploit...)
  • Web Scraper: Isolated browser environment
  • Sandboxed Execution: Docker containers
  • Automatic Tool Selection by task
  • OOB Attack Support via worker nodes

Agent Supervision

🎯 Smart Control
  • Execution Monitoring: Pattern detection & mentor intervention
  • Intelligent Task Planning: Automated 3-7 step decomposition
  • Tool Call Limits: Hard limits prevent runaway executions
  • Reflector Integration: Automatic error recovery
  • Optional Enhancement: 2x quality improvement for <32B models
04 · Quick Start

Get Up & Running in Minutes

System requires Docker, 2 vCPU, 4GB RAM, and 20GB disk space.

1
Download
Get installer for your OS
2
Configure
Set LLM provider & search
3
Deploy
docker-compose up
4
Attack
Start autonomous pentest
# Create directory & download installer mkdir -p pentagi && cd pentagi wget -O installer.zip https://pentagi.com/downloads/linux/amd64/installer-latest.zip unzip installer.zip # Run interactive installer sudo ./installer # The installer will: # 1. Verify Docker, network, and system requirements # 2. Configure .env with optimal defaults # 3. Set up LLM providers (OpenAI, Anthropic, Ollama...) # 4. Configure search engines (DuckDuckGo, Tavily...) # 5. Generate credentials & deploy with docker-compose
